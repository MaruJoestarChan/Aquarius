AgroSostenible MX - Landing page (HTML + CSS + JS)
---------------------------------------------------

Contenido:
- index.html
- styles.css
- script.js

Instrucciones rápidas:
1. Abrir index.html en cualquier navegador para ver la landing localmente.
2. Para publicar en Netlify: sube toda la carpeta al repositorio de GitHub o arrastra la carpeta al panel de 'Sites' en Netlify.
   - Netlify detecta sitios estáticos y desplegará index.html automáticamente.
   - El formulario usa 'data-netlify=true' para funcionar con Netlify Forms.

Personalización:
- Cambia textos en index.html.
- Reemplaza logo SVG por tu propio logo.
- Si quieres un formulario diferente, modifica el form en index.html.

Contacto de ejemplo en la página: contacto@agrosostenible.mx
